사용 시 주의

사용 가이드라인

✅ 올바른 사용

페이지당 하나의 Primary 버튼 사용
액션의 중요도에 따른 버튼 타입 선택
명확하고 액션 지향적인 라벨 사용
적절한 크기와 터치 영역 확보

❌ 피해야 할 사용

하나의 화면에 여러 Primary 버튼
모호한 라벨 ("확인", "OK")
너무 작은 터치 영역 (44px 미만)
일반 액션에 Danger 버튼 사용
